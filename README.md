## 💡 The Architecture

This project demonstrates a production-ready, minimalist "Apple/Stripe" aesthetic built for high-conversion AI startup waitlists. 

### Key UI Engineering Features:
* **Interactive Faux Terminal:** A purely CSS/HTML rendered chat mockup simulating a local LLM environment.
* **Cinematic Aesthetics:** Implemented noise overlays, massive spatial typography, and deep glassmorphism.
* **Radix Primitives:** Accessible, headless UI components perfectly styled for a premium dark-mode feel.

### 🛠 Tech Stack
* **Framework:** Next.js (React)
* **Styling:** Tailwind CSS
* **Components:** shadcn/ui (Radix)
* **Typography:** Geist / Plus Jakarta Sans
